## 🦀 Maryland Flash Card Quiz

A beginner-friendly Python command-line quiz that tests your knowledge of Maryland!

### Files

| File | Description |
|------|-------------|
| `flashquiz-maryland.py` | Main quiz program |
| `questions.json` | Flash card questions and answers |

### How to Run

Make sure you have Python 3 installed, then run from a terminal:

```bash
python3 flashquiz-maryland.py
```

Both files must be in the same folder when you run the quiz.

### What to Expect

- 10 fun questions about Maryland (history, sports, food, and more!)
- Type your answer and press **Enter**
- Get instant feedback after each question
- See your final score at the end

### Requirements

- Python 3 (no extra packages needed)
